# AI时代下的前端

## 为什么是JavaScript?

1. OpenAI / LangChain / dify 等都只提供Python和JavaScript的SDK
2. 全栈是趋势; Saas/Paas等隔离了服务器数据库等基础设施的复杂度
3. 唯一的全栈语言是 JavaScript


## 为什么是LangChain?

在构建复杂Chat Bot时, 聊天上下文/网络搜索/加载pdf等问题, 大模型框架可以帮助我们解决。

1. LangChain 有足够的流行度和认可度
2. LangChain.js 并不是 Python 版本的套壳，而是一个完整的团队从 0 开始构建的生态，足以看出官方对 JavaScript 生态的重视
